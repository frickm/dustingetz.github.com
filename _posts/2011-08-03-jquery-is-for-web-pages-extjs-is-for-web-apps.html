jquery is for web-pages, extjs is for web-apps 

 

jquery is for web pages 

ext is for web apps 

 

pages: like a blog, or a store like amazon, a typical .com 

apps: products, email client, or an mp3 player, or data views and manipulation 

 

ext provides a high level framework to think about web-apps in terms of ui widgets like pickers and grids. 

jquery provides a low level framework to think about web-pages in terms of raw html.
